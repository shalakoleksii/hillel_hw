### ДЗ 6. Міні-анкетування з підсумком 

Зробіть «анкету», яка збирає базову інформацію про користувача, а потім показує підсумок одним alert.

Через prompt послідовно запитайте:

Ім'я
Вік
Місто проживання
Чи любите Ви JavaScript? (користувач вводить "так" або "ні")
Після цього:

Збережіть значення в змінні примітивних типів.
Перетворіть відповідь про JavaScript в логічне значення:
якщо користувач ввів "так" (у будь-якому регістрі: "так", "Так", "ТАК") → вважаємо true
інакше → false
Сформуйте один рядок для alert, наприклад:
"Привіт, Іван! Вам 25 років, Ви з міста Київ. Ставлення до JavaScript: любить." або "... Ставлення до JavaScript: не любить."
Обов’язково:

Використати рядки та числа.
Використати if для перевірки відповіді про JS і підстановки відповідного тексту.
Використати конкатенацію рядків (+) або шаблонні рядки (якщо ви вже їх проходили — можна і без них, якщо рано).
